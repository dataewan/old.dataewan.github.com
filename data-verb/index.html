Data-verbs

  • Pandas assign function

    assign lets you put new columns on the dataframe, or assign new values to the old ones.

  • Pandas drop function

    drop lets you drop particular rows or columns of the dataframe.

  • Pandas groupby function

    This section of the documentation describes what the groupby function does groupby, but it is worth looking at the whole article as well. apply agg agg performs aggregations on your groupby object.

  • Pandas merge function

    merge is a lot like joining in SQL. It lets you combine together different datasets. The full documentation on merging is well worth a read. I’ll probably write about that at some other point.

  • Pandas pipe function

    Pipes in pandas

  • Pandas query function

    query performs filtering of the dataframe. You give it logical expressions, and it gives you back the rows that match that expression.

  • Pandas rename function

    rename changes the names of the columns in the dataframe.

  • Pandas sort_values function

    sort_values lets you order your dataframe.
